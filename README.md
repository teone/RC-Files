## .vimrc

Before using this `.vimrc` you should install:
- [vundle](https://github.com/gmarik/Vundle.vim)
- cmake

After pasting this file in your `$HOME`:
_Compile YCM_:
- `cd .vim/bundle/YouCompleteMe`
- `./install.sh`

_Install all the plugins_:
- Open vim
- `:PluginInstall`

And Enjoy!
